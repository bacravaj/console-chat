Работа с базой данный происходит через API MySQL.

Для запуска программы и проверки кода предварительно необходимо создать 
базу данных и необходимые таблицы(из файла "commands.txt").

Данная программа на данный момент не интегрирована в консольный чат из-за некоторых 
технических сложностей с реализацией серверного подключения.
