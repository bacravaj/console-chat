Работа с базой данный происходит через API MySQL.
Чат планировалось реализовать через клиен-серверную архитектуру. База данных развернута на сервере.
Подразумевается, что данный модуль должен считывать данные из базы данных и передавать их туда; основное же взаимодействие 
с консольным чатом должно осуществляться через функции из модуля "console-chat/server/TCP_server", который на данный момент не дописан.
Для запуска программы и проверки кода предварительно необходимо создать базу данных и необходимые таблицы(из файла "commands.txt").

Программа на данный момент не интегрирована в консольный чат из-за некоторых 
технических сложностей с реализацией серверного подключения.
